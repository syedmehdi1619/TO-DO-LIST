# 📝 To-Do List Web Application

A feature-rich single-page To-Do List web application built using **HTML, CSS, and JavaScript**. This project demonstrates the practical implementation of **Data Structures and Algorithms (DSA)** alongside modern web development concepts.

Features:
- ➕ Add, edit, and delete tasks
- 🔢 Automatic task indexing
- 🚩 Priority-based task management
- 🗑️ Deleted task history
- 🌙 Light & Dark Mode
- 📱 Responsive user interface
- ⚡ Fast and interactive single-page application

Data Structures & Algorithms Used:

 Array:Stores and manages all tasks. 
 Hash Table: Enables efficient task searching and indexing.
 Queue: Maintains tasks in First-In-First-Out (FIFO) order.
 Stack Stores: deleted tasks for delete history management. 
 Merge Sort: Sorts tasks efficiently based on selected criteria. 

Technologies Used:
- HTML5
- CSS3
- JavaScript (ES6)
